# Security Policy

## Reporting a Vulnerability

TRL Systems takes security seriously.

If you discover a vulnerability, please report it responsibly by contacting:

contact@trl-systems.com

Please include:

- Description of the issue
- Steps to reproduce
- Potential impact
- Suggested mitigation (if known)

We will respond within 5 business days.

## Security Measures
